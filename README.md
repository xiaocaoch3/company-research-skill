# Company Research Skill

An evidence-first Codex skill for company research and due diligence.

It helps structure investment analysis, job-seeker research, product and competitive analysis, and industry research. The workflow separates facts, analysis, and conclusions; prioritizes primary sources; tests competitive advantages against operating evidence; and makes uncertainty explicit.

## Contents

- `SKILL.md` — research principles, workflow, and output standard
- `agents/openai.yaml` — display metadata and default prompt

## Use

Install or copy this directory into your Codex skills directory, then invoke `$company-research` in a task.

## Example

```text
Use $company-research to analyze a company objectively for my investment decision.
```
